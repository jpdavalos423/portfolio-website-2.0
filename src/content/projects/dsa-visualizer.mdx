## Overview

DSA Visualizer is an educational project focused on making algorithm behavior easier to understand through animation and interaction.

## Problem

Students often struggle to build intuition for algorithms when they only see code or static diagrams.

## What I Built

An interactive visualizer that walks through sorting operations and highlights state changes across each step of execution.

## Technical Decisions

I prioritized clear visual feedback and incremental feature growth so new algorithms and structures can be added over time.

## Impact

The project serves as a practical learning tool while sharpening my front-end interaction design and algorithm communication skills.
