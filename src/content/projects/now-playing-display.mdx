## Overview

Now Playing Display is a small utility project that turns Spotify playback data into a real-time display for a dedicated device.

## Problem

I wanted a clean way to show current Spotify playback status on a Raspberry Pi display without manually refreshing anything.

## What I Built

A real-time now-playing interface that pulls Spotify playback information, manages authentication, and renders a display-friendly UI.

## Technical Decisions

I optimized the project for a constrained hardware display context and reliability of the Spotify auth flow.

## Impact

The project combined API integration, device-oriented UI thinking, and practical auth handling in a fun personal build.
