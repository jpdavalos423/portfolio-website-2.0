## Overview

Portfolio Website v1.0 was my first real attempt at designing, building, and deploying a personal project from scratch.

## Problem

I needed a place to present myself, document projects, and learn how production web pages are put together beyond tutorials.

## What I Built

A static personal site with sections for about, projects, experience, and contact, plus responsive styling and interactive UI polish.

## Technical Decisions

I used plain HTML, CSS/SCSS, and JavaScript to learn the fundamentals directly before moving to more structured frameworks.

## Impact

This project gave me the foundation to continue building more complex applications and iterate into later portfolio versions.
