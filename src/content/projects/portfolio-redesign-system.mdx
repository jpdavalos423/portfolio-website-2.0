## Overview

This project rebuilds a personal portfolio around clear positioning, strong case studies, and predictable content structure.

## Problem

The previous site mixed content styles and navigation patterns, making it harder for recruiters to scan relevant work quickly.

## What I Built

A content-driven Astro architecture with reusable components and route-based experience filtering.

## Technical Decisions

I optimized for static generation, strict scope control, and a lightweight UI foundation.

## Impact

The redesign improves maintainability while making future project additions faster.
