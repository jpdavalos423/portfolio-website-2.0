## Overview

PokeCollect is a collaborative web app for tracking a Pokemon card collection and viewing card information in one place.

## Problem

Collectors often need to cross-reference multiple sources to track cards and check market values.

## What I Built

A team-built interface for browsing cards, viewing details, and managing a personal collection backed by API-driven data.

## Technical Decisions

We used Agile workflows to break the project into iterative deliverables and integrated the Pokemon TCG API for live card data.

## Impact

The project strengthened my experience with team software development, planning, and API integration in a user-facing app.
